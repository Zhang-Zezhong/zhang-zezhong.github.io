# Zezhong Zhang - Personal Academic Website

This is the personal academic website of Zezhong Zhang (张泽中), Research Assistant Professor at CUHK Shenzhen.

## 🔗 Live Website
**https://zhang-zezhong.github.io/**

## 📋 About

This website showcases:
- Academic background and education
- Research interests (Edge Learning, Radio Map Sensing, B5G/6G Communications, Semantic Communications)
- Publications with citation metrics
- Contact information

## 🛠️ Technologies

- Pure HTML5 + CSS3 + Vanilla JavaScript
- Google Fonts (Inter, Noto Sans SC)
- Responsive design (mobile-friendly)
- No build tools required

## 📝 Update Publications

Edit `index.html` to update publications. The website uses a clean, academic style inspired by leading university research groups.

## 🚀 Deployment

The website is automatically deployed via GitHub Pages. Any push to the `main` branch will trigger a new deployment.
